# Synergy Light - Sitio Web Profesional

Sitio web multi-sección para **Synergy Light**, empresa de energía eléctrica residencial en Texas, EE.UU.

## 🌐 Sitio Web

**URL:** https://synergylight.com  
**Idioma:** Español (con opción Inglés)  
**Público:** Residentes hispanos de Texas

## 📋 Páginas Incluidas

| Página | Archivo | Descripción |
|--------|---------|-------------|
| **Inicio** | `index.html` | Página principal con hero, beneficios, planes, calculadora, testimonios, FAQ |
| **Nosotros** | `nosotros.html` | Historia, misión, visión, equipo, valores |
| **Servicios** | `servicios.html` | Energía residencial, proceso de conexión, soporte técnico |
| **Planes** | `planes.html` | Planes Básico, Estándar, Premium, tabla comparativa, calculadora |
| **Cobertura** | `cobertura.html` | Mapa de Texas, ciudades, verificación de código postal |
| **Testimonios** | `testimonios.html` | Opiniones de clientes, estadísticas de éxito |
| **FAQ** | `faq.html` | Preguntas frecuentes con buscador y categorías |
| **Contacto** | `contacto.html` | Formulario de contacto, cotización rápida, info de contacto |
| **Blog** | `blog/index.html` | Listado de artículos con búsqueda |
| **Artículo 1** | `blog/articulo-1.html` | 5 Consejos para Ahorrar Energía |
| **Artículo 2** | `blog/articulo-2.html` | Cómo Elegir el Plan de Energía Perfecto |
| **Artículo 3** | `blog/articulo-3.html` | Energía Solar en Texas: ¿Vale la Pena? |

## 🎨 Características de Diseño

- **Paleta de colores:** Verde neón (#00FF88) sobre fondos oscuros (#0A1628)
- **Tipografía:** Montserrat (títulos), Open Sans (cuerpo), Orbitron (precios/números)
- **Efectos:** Glow neon, partículas animadas, gradientes, animaciones scroll
- **Responsive:** Adaptado a móvil, tablet y desktop
- **Componentes:** Cards, modales, tabs, timeline, acordeones, sliders

## 🛠️ Tecnologías Utilizadas

- **HTML5** semántico con meta tags SEO y Open Graph
- **CSS3** con variables, flexbox, grid, animaciones keyframes
- **JavaScript** vanilla (ES6+) con Intersection Observer, módulos por función
- **JSON-LD** para datos estructurados (Organization, FAQ, LocalBusiness)
- **Font Awesome / Emojis** para iconografía

## ⚡ Funcionalidades JavaScript

- **Menú móvil** responsive con hamburguesa
- **Calculadora de ahorro** en tiempo real
- **Slider de testimonios** con autoplay
- **Acordeón FAQ** con búsqueda
- **Contadores animados** con Intersection Observer
- **Validación de formularios** con feedback visual
- **Partículas** animadas en hero
- **Lazy loading** de imágenes
- **Scroll spy** para menú activo
- **Compartir en redes sociales**

## 🚀 Instalación y Uso

1. Clona o descarga el repositorio
2. Abre `index.html` en tu navegador
3. Navega por todas las páginas desde el menú
4. Para desarrollo, edita los archivos en `assets/`

No requiere instalación de dependencias ni servidor. Funciona directamente en cualquier navegador moderno.
